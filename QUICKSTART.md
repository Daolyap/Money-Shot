# Quick Start Guide

## Installation

### Download and Run
1. Download the latest release from [GitHub Releases](https://github.com/Daolyap/Money-Shot/releases)
2. Extract the ZIP file to a folder of your choice
3. Run `MoneyShot.exe`

### First Launch
- The application will open the main window
- It will minimize to the system tray when closed
- Look for the Money Shot icon in your system tray

## Taking Your First Screenshot

### Method 1: Full Screen (Recommended for beginners)
1. Press `Print Screen` key on your keyboard
   - OR click "üì∏ Capture Full Screen" in the main window
2. The editor window will open with your screenshot
3. Click "üìã Copy" to copy to clipboard
   - OR click "üíæ Save" to save to a file

### Method 2: Select a Region
1. Press `Ctrl + Print Screen` on your keyboard
   - OR click "‚úÇÔ∏è Capture Region" in the main window
2. Your screen will dim with a dark overlay
3. Click and drag to select the area you want to capture
4. Release the mouse to capture

5. The editor will open with your selected region

## Saving Your Work

### Copy to Clipboard
1. Click "üìã Copy" button
2. Paste (Ctrl+V) into any application:
   - Word, PowerPoint, Photoshop
   - Email, Slack, Teams
   - Paint, GIMP, etc.

### Save to File
1. Click "üíæ Save" button
2. Choose where to save
3. Enter a filename (or use the auto-generated one)
4. Select format (PNG recommended)
5. Click Save

### Save to Both
1. Go to Settings (‚öôÔ∏è)
2. Select "Both clipboard and file"
3. Set your default save path
4. Now every save copies to clipboard AND saves a file!

## Using the System Tray

### Accessing Money Shot
1. Look for the Money Shot icon in your system tray (bottom-right)
2. Right-click the icon for the menu:
   - **Capture Full Screen**: Quick full capture
   - **Capture Region**: Quick region capture
   - **Settings**: Open settings
   - **Show Window**: Show the main window
   - **Exit**: Close Money Shot completely

### Quick Captures
- Double-click the tray icon to show the main window
- Use the right-click menu for instant captures
- Perfect for keeping Money Shot running in the background!

## Configuring Settings

### Opening Settings
- Click "‚öôÔ∏è Settings" in the main window
- OR right-click system tray icon ‚Üí Settings

### Essential Settings

#### Auto-Start with Windows
1. Check "Run on Windows startup"
2. Click Save
3. Money Shot will start automatically when Windows boots

#### Default Save Behavior
Choose one:
- **Clipboard only**: For quick sharing
- **File only**: For archiving
- **Both**: Best of both worlds (recommended!)

#### Custom Save Location
1. Click "Browse..." button
2. Select your preferred folder
3. All screenshots will save there by default

#### File Format
- **PNG**: Best quality, supports transparency (recommended)
- **JPG**: Smaller files, good for photos
- **BMP**: Uncompressed, largest files

## Common Tasks

### Create a Tutorial
1. Capture the relevant screen area

### Quick Note

4. Paste into OneNote/Evernote/etc.

### Share a Funny Moment
1. Capture the screen
2. Add annotations to emphasize the funny part
3. Copy to clipboard
4. Paste into chat/social media

## Troubleshooting

### Screenshot is Black
- Some applications use protected rendering (DRM)
- Try capturing a different application
- Or use Windows Snipping Tool for protected content

### Hotkeys Don't Work
- Another application might be using the same hotkey
- Check for other screenshot tools running
- Restart Money Shot

### Can't Find My Screenshots
- Check Settings ‚Üí Default save path
- Default is usually: Pictures folder
- Make sure "File" or "Both" is selected

### App Disappeared
- Check system tray (bottom-right corner)
- Right-click Money Shot icon
- Click "Show Window"

### App Won't Start on Boot
- Check Settings ‚Üí "Run on Windows startup"
- Make sure it's checked
- Click Save
- Restart Windows to test
